## ðŸš€ How to run
```bash
  # cloning repository
  git clone https://github.com/AugustoMarcelo/marvel-heroes-challenge

  # access folder
  cd marvel-heroes-challenge

  # install dependencies
  yarn

  # enable reverse proxy (allow you to use localhost)
  adb reverse tcp:3000 tcp:3000

  # running fake api
  yarn api

  # install app
  yarn android
```
